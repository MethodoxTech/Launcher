Launcher

A fast, non‑intrusive, self‑contained “task bar” for launching executables, folders, and URLs.

View on GitHub Download on itch.io

What is Launcher?

Think of Launcher as a mini‑Rainmeter gadget—but way faster and non‑intrusive once you add it to your PATH. Use it to open executables with arguments, reveal files/folders, or launch websites—all from a simple CLI or the “Big White Dot” GUI.

Features

  • Launch .exe's with custom arguments
  • Open folders or select files in Explorer
  • Open URLs in your default browser
  • Big White Dot GUI

Setup

Make sure you have a default editor associated with .yaml files, or Windows may show “Application Not Found” errors. If you use VS Code, point your PATH at: 

%UserProfile%\AppData\Local\Programs\Microsoft VS Code

(Not the default “bin” folder, which only contains helper scripts.)
